Description:
META is a Fullstack Website made with Laravel Framework. It allows users to record and track and categorize their expenses for better financial practices. It also has helpful articles that can be saved for future readings and unsave when necessary. Upon registration, new users will receive a verification email. Verified users has access to better features than unverified users. CRUD Operations for Admin is available for articles and user settings.
Features:
  • Admin and User Interfaces
  • Register and Log In
  • Data Validation (AJAX and Laravel)
  • Email Verification (Laravel)
  • Page/Feature Restrictions base on User Role and Verification
  • CRUD Operations for both Admin and User
  • Content Writing directly on webpage
  • Image Preview before Upload
  • Multiple category on one Article
  • Soft Deletes for Article Deletion
  • Mobile Responsive Design
